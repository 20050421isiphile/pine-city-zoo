# Pine City Zoo Website

A multi-page web application built to display 
information about Pine City Zoo.

## Features
- Interactive zoo map
- Animal information pages
- Places to visit
- Dining, weather and feedback pages

## Built With
HTML, CSS

## What I learned
- Writing clean, organised code
- Building multi-page websites
- Version control using Git and GitHub

## Purpose
Personal project to develop web development skills
in preparation for a software development career.
